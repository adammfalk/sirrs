## USA Referee Resources

USA Rugby <a href='https://usa.rugby/referee-resources' target='_blank'>Referee Resources</a> has some fantastic resources like Game Management Guidelines, links to important
World Rugby clarifications, local USA Rugby policies, and game day type information.
It gets updated with some frequency.

### Game Management Guidelines

Find the 2025-2026 Game Management Guidelines <a href='https://d26phqdbpt0w91.cloudfront.net/NonVideo/4abafdf7-930a-44e7-94ad-6b4fd248f81a.pdf' target='_blank'>here</a>.

### Protective Equipment & Clothing Guidelines

Current guidance, with lots of wonderful pictures, can be found <a href='https://d26phqdbpt0w91.cloudfront.net/NonVideo/d32ef3b6-46e5-4c06-9fc5-449bfb2f2963.pdf' target='_blank'>here</a>. This is used to help provide consistent guidance on whether or not clothing/padding/saftey devices are law compliant.

## CBT courses

USA Rugby's Learning Center has <a href='https://usarugby.docebosaas.com/learn/catalog' target='_blank'>a number of webinars</a> saved ranging on topics of diet, stretching, speed work, refereeing youth, foul play, and a number of other topics specific to refereeing.

:::tip
The Assistant Referee Certification course is on USA Rugby's Learning Center
:::
